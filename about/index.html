学习记录

  1. 进入node.js官网（https://nodejs.org/en/）下载安装包
  2. 使用指令（npm install vue-cli -g(全局)）安装vue-cli（有效的链接资源 https://www.cnblogs.com/blog-cxj2017522/p/8525902.html）
  3. 安装hexo，配置npm中国全量景象
  4. 安装完成后 使用hexo init 新建项目目录，随后进入目录 cd 接着编译目录 npm install
  5. 学会使用hexo new 指令
  6. 在hexo官网上找hexo主题，然后学会使用主题
  7. hexo和markdown关联起来
